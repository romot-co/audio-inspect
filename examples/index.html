技術仕様

Audio InspectはWebブラウザで動作するリアルタイム音声解析ライブラリです。

Web Audio API AudioWorklet TypeScript Canvas 2D ES Modules

最新のWeb標準技術を使用し、プラグインやライブラリのインストールなしで 高品質な音声解析機能を提供します。

📱 GitHub リポジトリ

使用方法

  1. マイクアクセスを許可
    ブラウザがマイクアクセスを要求した際に「許可」を選択してください。
  2. デモページを選択
    上記のカードから興味のある解析機能を選んでクリックしてください。
  3. 解析開始
    各デモページで「解析開始」ボタンを押すと、リアルタイム解析が開始されます。
  4. 結果を確認
    グラフやメーターで音声特徴量をリアルタイムで確認できます。

対応ブラウザ

Chrome 76+ Firefox 76+ Safari 14.1+ Edge 79+

AudioWorkletをサポートする最新のWebブラウザが必要です。
最適な体験のためにはChrome、Firefox、Safariの最新版をお使いください。
